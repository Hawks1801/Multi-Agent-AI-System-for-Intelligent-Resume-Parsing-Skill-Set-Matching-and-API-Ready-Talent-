## 🛠️ Unique Agentic Features

### 1. AI Interview Strategist
Our system doesn't just match resumes; it prepares you for the interview. The **Interview Agent** analyzes the gap between candidate skills and job requirements to generate:
- **Verification Questions:** To probe deep intoclaimed strengths.
- **Gap Bridging Questions:** To assess the candidate's adaptability in missing areas.
- **Strategic Rationale:** AI-driven reasoning for the chosen interview tactic.

### 2. Hierarchical Skill Taxonomy (5,000+ Skills)
Unlike simple keyword matchers, our **Normalizer Agent** uses a deep, hierarchical database covering 10+ industries. It intelligently maps variations (e.g., "K8s" -> "Kubernetes") using semantic vector embeddings.

### 3. Agentic Orchestration via LangGraph
The system uses a directed cyclic graph (DAG) to orchestrate agents. This ensures that errors in one agent (e.g., Parser) are gracefully handled and logged without crashing the entire pipeline.
